## 阶段零：量化思维（1周）

> 目标：理解量化的本质，知道它帮你解决什么问题

**要学的内容：**
- 什么是量化交易：用工厂流水线类比
  - 手工交易 = 街边小摊，全凭老板手感
  - 量化交易 = 工厂流水线，每个环节标准化、可测试、可优化
- 量化交易的四大支柱：
  - 1️⃣ **信号** — 什么时候买/卖
  - 2️⃣ **执行** — 如何下单、滑点控制
  - 3️⃣ **风控** — 止盈止损、仓位管理
  - 4️⃣ **评估** — 这个策略到底赚不赚钱
- 量化的局限：黑天鹅、过拟合、幸存者偏差

**我帮你做的事：** 用你熟悉的工厂例子讲解每个概念，没有公式，只有直觉。

- **交易逻辑**：开仓条件、平仓条件、过滤条件 — 用订单工单类比

## 阶段四：回测与评估（2-4周）

> 目标：不再凭感觉判断策略好不 好，而是用数据说话

**核心知识：**
- 绩效指标含义 & 怎么用（每个都有工厂比喻版）：
  - **总收益率** — 这个季度工厂赚了多少钱
  - **年化收益率** — 把赚的钱折合成一年的利润率
  - **最大回撤** — 工厂最困难的时候亏了多少
  - **夏普比率** — 每承担一份风险能赚多 少回报
  - **胜率 & 盈亏比** — 做10笔赚几笔，赚的钱能不能覆盖亏的
- 基准对比：你的策略跑赢 SPY 了吗？
- 过拟合：别把历史数据背下来当策略

**我帮你做的事：**
- 写完整的回测框架代码
- 每个指标用数据+图表解释
- 你的策略跑完，我帮你写绩效报告

## 阶段五：策略进阶（4-8周）

> 目标：能独立设计、优化、量化策略

**进阶方向（选你感兴 趣的）：**

### 方向A：多因子选股
```
基本面因子 + 技术面因子 + 情绪因子
→ 打分排名 → 选前N只股票
```
- 因子构建：PE、ROE、动量、波动率
- 因子组合：等权、最优化、机器学习权重
- 代码量：~200行

### 方向B：期权量化策略
```
轮动 Covered Call、Cash-Secured Put 
期权平价、波动率交易
```
- 期权组合回测、希腊字母监控
- 波动率曲面分析
- 代码量：~150行

### 方向C：组合与资金管理
```
风险平价、全天候组合、Kelly公式
投资组合优化（均值-方差/Monte Carlo）
```
- 代码量：~100行

### 方向D：机器学习入门
```
简单线性回归预测 → 随机森林选股 → 
LSTM价格预测（纯学习，不实盘推荐）
```
- 代码量：~100行
