## 📊 2. 帮你做回测 <a id="backtest"></a>

**场景：** 你有一个策略思路，想看历史表现怎么样。

**我能做的回测类型：**

| 策略类型 | 实例 |
|----------|------|
| 📈 趋势跟踪策略 | 均线交叉、MACD、布林带突破 |
| 🏷️ 均值回归策略 | RSI 超买超卖、布林带回归、配对交易 |
| 📉 动量策略 | 过去 N 日收益率排名、板块轮动 |
| 🛡️ 组合策略 | 股债配置、风险平价、全天候组合 |
| 📊 期权策略 | Covered Call 滚动、备兑开仓增强收益 |

**回测报告包含：**
```
✅ 总收益率 & 年化收益率
✅ 夏普比率 / 索提诺比率 / 卡玛比率
✅ 最大回撤 & 回撤持续时间
✅ 胜率 & 盈亏比
✅ 交易次数 & 平均持仓天数
✅ 资金曲线图 & 回撤图
✅ 与基准（如 SPY）对比
```

> 🎯 我能帮你对回测结果做归因分析，告诉你哪里赚钱、哪里亏钱。

## 🧠 3. 帮分析 & 优化策略 <a id="analysis"></a>

**场景：** 你有一个策略，或刚跑完回测，想看看能不能改进。

**我能做：**
- **参数优化** — 网格搜索 / 随机搜索 / 贝叶斯优化最优参数
- **过拟合检测** — 蒙特卡洛模拟、交叉验证、随机化测试
- **添加风控** — 止损止盈、移动止损、仓位缩减规则
- **多因子融合** — 基础面 + 技术面 + 情绪面的多信号叠加
- **资金管理** — Kelly formula、固定比例、风险平价仓位建议
- **压力测试** — 模拟 2008/2020 等极端行情下的策略表现
