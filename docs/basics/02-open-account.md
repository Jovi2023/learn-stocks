## 第一步：选券商

国内投资者目前主流选择就两个：

## 现实情况——盈透证券（IB）已经开不了

**2025年8~9月开始，盈透证券收紧中国大陆居民开户政策。** 现在开户必须提供**海外居住或工作证明**（如境外签证、境外居住地址、境外工作合同等），纯中国大陆身份证/地址已经无法通过审核。

这意味着目前国内投资者能开的美股券商只剩下：

### 嘉信理财（Schwab）——中国大陆首选

**⚠️ 关于嘉信理财的 $25,000 门槛**

嘉信理财官网确实写国际账户最低 $25,000，但那是指**香港分公司**。中国大陆投资者可以申请 **嘉信国际账户（Schwab International）**，走独立入口，**没有最低入金要求**。

开户地址：[https://international.schwab.com/](https://international.schwab.com/)

## 第二步：开户需要什么

**无论选哪个，都需要准备：**

- 📄 身份证或护照
- 📄 地址证明（近3个月的信用卡账单或银行流水，有英文更佳）
- 📄 税务信息（中国税务居民，填W-8BEN表——后面专门讲）
- 💳 一个香港或海外银行账户（推荐）或国内银行账户（电汇用）

> ⚠️ 嘉信理财审核相对严格，材料一定要真实清晰，否则会被反复补件。但整体来说，中国大陆居民目前**嘉信理财是最友好、最可行的选择**。

## 第三步：入金（把钱转进去）

这是最麻烦的一步，也是绕不开的。

### 方案一：国内银行电汇（最传统）

| 步骤 | 说明 |
|------|------|
| 1. 购汇 | 在银行App购汇（如中国银行、招商银行），选"美元" |
| 2. 跨境汇款 | 填写券商给你的收款账户信息（SWIFT码、账号、银行地址） |
| 3. 到账 | 3~5个工作日，手续费约150~300元/笔 |

**注意：**
- 每人每年有 **$50,000 外汇额度**
- 汇款用途不要写"投资"，建议写"生活费"或"旅游"
- 2026年新规：单笔 ≥ $1,000 的境外汇款须额外申报

### 方案二：香港银行账户（推荐）

如果有香港银行卡（或去香港办一张），入金会顺畅很多：

```
国内人民币 → 购汇成美元 → 汇到香港银行 → 香港银行 → FPS/转账到券商
```

香港银行入金到盈透证券，通过**转数快(FPS)**可以半小时内到账，手续费极低甚至免费。

### 方案三：嘉信理财借记卡（最方便）

嘉信理财会给你一张 VISA 借记卡。**全球ATM直接取当地货币，免手续费，不占外汇额度。**

也就是说：你的美股赚了钱 → 用嘉信借记卡在国内ATM直接取出人民币 → 完全合规。

---

## 第四步：税务——别被吓到

作为中国税务居民（没有美国绿卡/护照），你只需要：

**开户时填一张 W-8BEN 表**（券商全程指导），声明你是外国人，主要好处：

- ✅ 股息税率从30%降到 **10%**
- ✅ 资本利得（买卖差价）**完全不用交美国税**
- ✅ 在中国也不用交资本利得税（目前中国政策）

**你只需要注意：** 中国有"全球征税"的说法，但目前个人炒美股基本没人主动申报，实操中也没被追缴。但如果你想完全合规，建议咨询税务专业人士。
