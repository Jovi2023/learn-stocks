## 第一步：选券商

国内投资者目前主流选择就两个：

## 现实情况——盈透证券（IB）已经开不了

**2025年8~9月开始，盈透证券收紧中国大陆居民开户政策。** 现在开户必须提供**海外居住或工作证明**（如境外签证、境外居住地址、境外工作合同等），纯中国大陆身份证/地址已经无法通过审核。

这意味着目前国内投资者能开的美股券商只剩下：

### 嘉信理财（Schwab）——中国大陆首选

**嘉信理财（Charles Schwab）是谁？**

- 美国最大券商之一，成立于1971年，至今超过50年历史
- 纳斯达克上市公司（股票代码：SCHW），市值约 **$1,290亿**
- 管理客户资产超过 **$9万亿**（是盈透证券的4倍多）
- 受美国 **SEC**（证券交易委员会）和 **SIPC**（证券投资者保护公司）监管
- 账户受 **SIPC 最高 $50万美元保险**保护

**它不只是券商，还是一家银行。** 所以它有：股票交易平台 + 借记卡 + 支票本 + 货币基金 + 美国国债和企业债购买，功能远不止炒股。

**有独立的股票交易平台吗？** ✅ 有。开户后可以在 **schwab.com 网页端** 或 **嘉信手机App** 上直接交易股票、ETF和期权，界面有中文版，也有中文客服电话（国内可直拨）。

**⚠️ 关于嘉信理财的 $25,000 门槛**

嘉信理财官网确实写国际账户最低 $25,000，但那是指**香港分公司**。中国大陆投资者可以申请 **嘉信国际账户（Schwab International）**，走独立入口，**没有最低入金要求**。

## 第二步：开户需要什么

**无论选哪个，都需要准备：**

- 📄 身份证或护照
- 📄 地址证明（近3个月的信用卡账单或银行流水，有英文更佳）
- 📄 税务信息（中国税务居民，填W-8BEN表——后面专门讲）
- 💳 一个香港或海外银行账户（推荐）或国内银行账户（电汇用）

> ⚠️ 嘉信理财审核相对严格，材料一定要真实清晰，否则会被反复补件。但整体来说，中国大陆居民目前**嘉信理财是最友好、最可行的选择**。

## 第三步：入金（把钱转进去）

这是最麻烦的一步。目前中国投资者入金嘉信有三种主流方式，按推荐程度排序：

### 方案一：Wise 入金（推荐，手续费最低）

**Wise（原 TransferWise）** 是一个跨境转账平台，支持你以极低成本把人民币换成美元，然后通过 **ACH（美国本地转账）** 方式免手续费转入嘉信。

**流程：**

```
国内银行卡 → 人民币 → Wise → 美元 → 嘉信账户（ACH到账，0手续费）
```

**具体步骤：**

| 步骤 | 操作 |
|------|------|
| 1 | 注册 Wise 账号（wise.com，有中文） |
| 2 | 在 Wise 内开通"美元账户"，你会得到一个美国本地 Routing Number 和 Account Number |
| 3. | 将 Wise 的美元账户绑定到嘉信（嘉信APP → Transfer Funds → External Accounts → Add Account → 填入Wise的路由号和账号） |
| 4 | 在嘉信APP发起 ACH 转账：嘉信主动从 Wise "拉" 钱进来（From: Wise, To: 嘉信） |
| 5 | 等待1~3个工作日清算完成，资金可自由交易 |

**费用：** Wise 换汇手续费约 0.5%~1%，远低于银行电汇。ACH到嘉信部分 **0手续费**。

**注意：** ACH入金后有几个交易日"清算期"，期间不能交易期权/期货，但可以正常买卖股票。

### 方案二：国内银行电汇（最传统）

从国内银行直接电汇美元到嘉信的收款账户。

| 步骤 | 说明 |
|------|------|
| 1. 购汇 | 在银行App购汇（推荐中行、招商），选"美元" |
| 2. 跨境汇款 | 填嘉信提供给你的收款账户信息（SWIFT码、账号、银行地址） |
| 3. 到账 | 3~5个工作日，手续费约150~300元/笔 |

**注意：**
- 每人每年 **$50,000 外汇额度**
- 汇款用途不要写"投资"，建议写"生活费"或"旅游"
- 2026年新规：单笔 ≥ $1,000 的境外汇款须额外申报
- 部分银行可能会拦截"证券"相关的汇款，需要提前打电话确认

### 方案三：香港银行账户（进阶方案）

如果你有香港银行账户（或计划去香港办理），流程更顺畅：

```
国内人民币 → 购汇 → 汇到香港银行账户 → 香港银行 → 电汇到嘉信
```

好处是香港汇款没有每年$50,000限制，到账也更快。

### 取现：嘉信借记卡（全球免费）

嘉信会给你一张 VISA 借记卡，**全球任何ATM取当地货币都免手续费，不占外汇额度。**

**也就是说：** 美股赚了钱 → 用嘉信借记卡在国内ATM直接取出人民币 → 完全合规、不占用5万额度。这是嘉信相比其他券商最大的优势。

---

## 第四步：税务——别被吓到

作为中国税务居民（没有美国绿卡/护照），你只需要：

**开户时填一张 W-8BEN 表**（券商全程指导），声明你是外国人，主要好处：

- ✅ 股息税率从30%降到 **10%**
- ✅ 资本利得（买卖差价）**完全不用交美国税**
- ✅ 在中国也不用交资本利得税（目前中国政策）

**你只需要注意：** 中国有"全球征税"的说法，但目前个人炒美股基本没人主动申报，实操中也没被追缴。但如果你想完全合规，建议咨询税务专业人士。
