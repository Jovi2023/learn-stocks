## K线（蜡烛图）长什么样

每根K线记录了**开盘价、收盘价、最高价、最低价**四个价格：

```
      最高价 ───┬───
                │
    ┌───────────┤
    │  实体     │  ← 红色 = 收盘 > 开盘（上涨）
    │  (阳线)   │     绿色 = 收盘 < 开盘（下跌）
    └───────────┤
                │
      最低价 ───┴───
```

> 💡 美股软件常用 **绿涨红跌**（和A股相反），但也有可以调颜色的，按你习惯来。

**一根K线看不出来什么，但一排K线能告诉你：**
- 趋势（在涨还是在跌）
- 波动（上下影线长 = 波动大）
- 力度（实体大 = 买卖决心强）

## 三个最常用的指标

太多的指标会让你迷路。新手先看这三个就够了：

### 1. 移动平均线（MA）

**是什么：** 过去N天收盘价的平均值，连成一条线。

| 周期 | 叫什么 | 代表什么 |
|------|--------|---------|
| 20天 | MA20 | 短期趋势 |
| 50天 | MA50 | 中期趋势 |
| 200天 | MA200 | **牛熊分界线** |

**怎么用：**
- 股价在MA200上方 → 长期趋势向上
- 股价在MA200下方 → 长期趋势向下
- MA20上穿MA50/MA200 → **金叉**，看涨信号
- MA20下穿MA50/MA200 → **死叉**，看跌信号

> **用工厂打比方：** 移动平均线就像你工厂过去一段时间的**平均接单量**。单日订单突然多了（单日大涨）可能只是巧合，但如果连续一个月都高于平均值（股价站上均线），那就是真的大客户来了。

### 2. 相对强弱指标（RSI）

**是什么：** 衡量股票"超买"还是"超卖"，范围 0~100。

| RSI 值 | 含义 |
|--------|------|
| > 70 | **超买**，可能涨太多要回调了 |
| 30~70 | 正常区间 |
| < 30 | **超卖**，可能跌过头要反弹了 |

**怎么用：**
- RSI > 70 → 考虑分批卖出
- RSI < 30 → 考虑买入机会

**注意：** 强势股可以在RSI>70后继续涨很久，不要光靠这个做买卖决定，把它当"提醒信号"。

### 3. MACD（指数平滑异同移动平均线）

**是什么：** 两条快慢均线的差值，加上一条信号线。

**怎么用：** 只看最关键的一点：
- MACD线上穿信号线（0轴下方）→ **买入信号**
- MACD线下穿信号线（0轴上方）→ **卖出信号**

新手不用深究它的算法，把它当作"趋势加速器"就行。

## 你应该打开什么周期看？

| 交易类型 | 主看周期 |
|---------|---------|
| 做长线（持有数月/年） | **日线、周线** |
| 做波段（持有数周） | **日线** |
| 做短线（持有数天） | **60分钟线、日线** |
| 做日内（当天进出） | **5分钟、15分钟** |

**新手建议：** 看日线 + 周线，忽略分钟级波动。你不是来盯盘的。

## 一张图就能做判断

```
打开一只股票的日线图，依次问自己：

1. 股价在 MA200 上面还是下面？        → 判断大趋势
2. MA20 和 MA200 的关系是金叉还是死叉？ → 判断趋势方向
3. RSI 是多少？                       → 判断是否过热或过冷
4. MACD 在零轴上还是零轴下？          → 判断动量
```

这四个问题看完了，你对这只股票就有数了。

---

## 一句话总结

> 均线看趋势，RSI看是否过热，MACD看动量。
> 三个指标结合起来用，别只看一个就下结论。
