# 第5篇：如何看财报——抓关键

> 日期：2026-05-18 | 状态：初稿

---

财报看起来吓人（动辄上百页），但你**不需要都看完**。抓住几个关键数字就够了。

更重要的是：**别等财报发出来再去分析，那时股价已经反应完了。你应该在财报前就了解这些指标，用来判断这家公司值不值得持有。**

---

## 三个最重要的数字

### 1. 每股收益（EPS）

```
EPS = 净利润 ÷ 总股数
```

**它代表了什么：** 公司每1股赚了多少钱。

**用工厂打比方：** 你家户外家具厂今年净利润100万，分成100份（相当于总股本），每份赚1万。这就是EPS。

**怎么看：**
- EPS **正数且增长** → 公司赚钱，且在赚更多
- EPS **负数** → 公司在亏钱
- 看连续3~5年的EPS增长趋势，而不是只看一个季度

### 2. 市盈率（PE）

```
PE = 当前股价 ÷ EPS
```

**它代表了什么：** 按目前利润，市场愿意出多少倍的价格买这家公司。

**用工厂打比方：** 你家工厂每年赚100万，现在有人出价1000万买你的厂。PE = 1000万÷100万 = 10倍。意思是"10年回本"。

**怎么看：**

| PE 范围 | 含义 |
|---------|------|
| < 15 | 可能被低估，也可能是夕阳行业 |
| 15~25 | 正常合理区间 |
| 25~50 | 成长型公司，市场看好它未来高增长 |
| > 50 | 非常贵，除非增长极其强劲，否则风险很高 |

**对比同行业更有效：** 一家科技公司PE 30不算高，一家银行PE 30算很高。

### 3. 营收增长率（Revenue Growth）

```
营收增长率 = (本期营收 - 上期营收) ÷ 上期营收 × 100%
```

**它代表了什么：** 公司卖东西的规模在长大还是缩小。

**用工厂打比方：** 去年做了1000万的订单，今年做了1200万 → 增长20%。

**为什么重要：** 公司可以靠砍成本"造"出利润（短期有效），但营收增长才是真正的成长。**没有营收增长的利润增长不可持续。**

---

## 看利润率的三个层次

| 指标 | 含义 | 健康范围（一般制造业） |
|------|------|---------------------|
| **毛利率** | 扣除直接成本后剩多少 | > 30% 算好 |
| **营业利润率** | 扣除所有运营费用后剩多少 | > 10% 算正常 |
| **净利润率** | 交完税、还完利息后净赚多少 | > 5% 算及格 |

**你的家具厂参考：** 原材料（铝合金、柚木）涨了 → 毛利率就跌。毛利率稳不住，后面两层也守不住。

---

## 一看就懂的财报检查清单

季度/年度财报出来后，按顺序看这5个数字：

```
□ 营收：同比（和去年同期比）增长了吗？增长了多少？
□ 每股收益(EPS)：增长了吗？是否超出市场预期？
□ 毛利率：稳定还是在下降？
□ 自由现金流：公司账上现金在变多还是变少？
□ 管理层展望：下季度预期是多少？和市场预期一致吗？
```

> 💡 **市场最在意的是"预期差"：** 实际EPS比分析师预期高 → 股价大概率涨。低 → 大概率跌。公司自己"好但没预期好"，也会跌。

---

## 一句话总结

> 看财报不追细节，抓住 EPS、PE、营收增长率 三个数。
> 好公司 = 营收稳定增长 + EPS持续提升 + PE合理。
