# 第5篇：如何看财报——抓关键

> 日期：2026-05-18 | 状态：初稿

---

财报看起来吓人（动辄上百页），但你**不需要都看完**。抓住几个关键数字就够了。

更重要的是：**别等财报发出来再去分析，那时股价已经反应完了。你应该在财报前就了解这些指标，用来判断这家公司值不值得持有。**

---

## 三个最重要的数字

### 1. 每股收益（EPS）

```
EPS = 净利润 ÷ 总股数
```

**它代表了什么：** 公司每1股赚了多少钱。

**用工厂打比方：** 你家户外家具厂今年净利润100万，分成100份（相当于总股本），每份赚1万。这就是EPS。

**怎么看：**
- EPS **正数且增长** → 公司赚钱，且在赚更多
- EPS **负数** → 公司在亏钱
- 看连续3~5年的EPS增长趋势，而不是只看一个季度

### 2. 市盈率（PE）

```
PE = 当前股价 ÷ EPS
```

**它代表了什么：** 按目前利润，市场愿意出多少倍的价格买这家公司。

**用工厂打比方：** 你家工厂每年赚100万，现在有人出价1000万买你的厂。PE = 1000万÷100万 = 10倍。意思是"10年回本"。

**怎么看：**

| PE 范围 | 含义 |
|---------|------|
| < 15 | 可能被低估，也可能是夕阳行业 |
| 15~25 | 正常合理区间 |
| 25~50 | 成长型公司，市场看好它未来高增长 |
| > 50 | 非常贵，除非增长极其强劲，否则风险很高 |

> ⚠️ **一定要跟同行比。** 一家科技公司PE 30不算高，一家银行PE 30算很高。

#### 各行业合理的 PE 范围

下表基于 2026 年最新 S&P 500 行业数据，列出各行业 TTM（过去12个月）滚动市盈率的典型范围：

| 行业 | 典型 PE 范围 | 代表公司 | PE为什么高/低 |
|------|:-----------:|---------|-------------|
| **信息技术/软件**（含 AI/半导体） | 30~45 | 英伟达(NVDA)、微软(MSFT)、苹果(AAPL) | 高增长预期，AI 浪潮推动，市场愿意为未来利润付高价 |
| **可选消费**（电商/汽车/零售） | 25~35 | 亚马逊(AMZN)、特斯拉(TSLA)、家得宝(HD) | 与经济周期关联大，成长性强的如亚马逊PE较高，传统零售较低 |
| **通信服务**（社交媒体/娱乐） | 20~30 | 谷歌(GOOGL)、Meta(META)、奈飞(NFLX) | 广告收入驱动，增长稳定，PE中等偏上 |
| **医疗保健** | 20~28 | 强生(JNJ)、辉瑞(PFE)、联合健康(UNH) | 防御性行业，盈利稳定但增长有限，PE温和 |
| **工业** | 20~28 | 卡特彼勒(CAT)、波音(BA)、霍尼韦尔(HON) | 周期性强，受制造业景气度影响大 |
| **房地产**（REITs） | 30~40 | 安博(PLD)、Equinix(EQIX) | PE偏高是因为折旧会计压低账面利润，实际现金流更好，更适合看FFO（营运现金流） |
| **日常消费品**（必选消费） | 18~25 | 可口可乐(KO)、宝洁(PG)、沃尔玛(WMT) | 刚需行业，增长慢但稳定，PE适中 |
| **公用事业** | 15~22 | 杜克能源(DUK)、南方公司(SO) | 增速极低(1~3%/年)，但股息稳定，类似债券 |
| **金融/银行** | 12~18 | 摩根大通(JPM)、高盛(GS)、伯克希尔(BRK.B) | 利率敏感型，盈利周期性波动大，市场不愿给高PE |
| **能源**（石油/天然气） | 10~18 | 埃克森美孚(XOM)、雪佛龙(CVX) | 强周期，油价一跌利润就崩，市场给最低PE |
| **基础材料**（化工/金属/采矿） | 12~20 | 陶氏化学(DOW)、自由港(FCX) | 受大宗商品价格影响大，周期性明显 |

**用你的家具厂来理解：**

假设你的家具厂年利润稳定100万：
- 如果别人出价 **1000万** 买你的厂（PE=10），相当于他们觉得家具行业增长空间有限
- 如果别人出价 **4000万** 买（PE=40），相当于他们认为你工厂有独家技术、能大幅扩张

**关键原则：**
1. **高PE不一定贵**——如果利润增速快，高PE很快就会被追赶（这叫"PEG"指标，后面会讲）
2. **低PE不一定便宜**——可能是行业在走下坡路，利润会不断缩水
3. **永远跟同行业比**——拿银行PE和软件公司比，没有意义

> 📌 数据来源：S&P 500 各行业板块2026年5月TTM PE数据。

### 3. 营收增长率（Revenue Growth）

```
营收增长率 = (本期营收 - 上期营收) ÷ 上期营收 × 100%
```

**它代表了什么：** 公司卖东西的规模在长大还是缩小。

**用工厂打比方：** 去年做了1000万的订单，今年做了1200万 → 增长20%。

**为什么重要：** 公司可以靠砍成本"造"出利润（短期有效），但营收增长才是真正的成长。**没有营收增长的利润增长不可持续。**

---

## 看利润率的三个层次

| 指标 | 含义 | 健康范围（一般制造业） |
|------|------|---------------------|
| **毛利率** | 扣除直接成本后剩多少 | > 30% 算好 |
| **营业利润率** | 扣除所有运营费用后剩多少 | > 10% 算正常 |
| **净利润率** | 交完税、还完利息后净赚多少 | > 5% 算及格 |

**你的家具厂参考：** 原材料（铝合金、柚木）涨了 → 毛利率就跌。毛利率稳不住，后面两层也守不住。

---

## 一看就懂的财报检查清单

季度/年度财报出来后，按顺序看这5个数字：

```
□ 营收：同比（和去年同期比）增长了吗？增长了多少？
□ 每股收益(EPS)：增长了吗？是否超出市场预期？
□ 毛利率：稳定还是在下降？
□ 自由现金流：公司账上现金在变多还是变少？
□ 管理层展望：下季度预期是多少？和市场预期一致吗？
```

> 💡 **市场最在意的是"预期差"：** 实际EPS比分析师预期高 → 股价大概率涨。低 → 大概率跌。公司自己"好但没预期好"，也会跌。

## 一句话总结

> 看财报不追细节，抓住 EPS、PE、营收增长率 三个数。
> 好公司 = 营收稳定增长 + EPS持续提升 + PE合理。
