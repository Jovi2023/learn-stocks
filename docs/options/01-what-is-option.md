# 第11篇：期权是什么——用最朴素的方式理解

> 日期：2026-05-18 | 状态：初稿

---

期权的概念让很多人望而却步，但它本质上很简单。

## 用你家工厂来打比方

你有一个客户想预订一批户外躺椅，但不确定3个月后是否需要。

**你和客户签了一个"权利协议"：**

> 客户今天付给你 **$500（权利金）**
> 他有权在未来3个月内，**以$10,000的价格** 买你这一批躺椅
> 如果他最后不想买了，**最多损失这$500**

**这就是期权。**

- 客户 = 期权买方
- 你 = 期权卖方
- $500 = **权利金（Premium）**
- $10,000 = **行权价（Strike Price）**
- 3个月 = **到期日（Expiration）**

---

## Call 和 Put

期权分两种，记住这两个定义就够了：

| 期权类型 | 通俗理解 | 你赌什么 |
|---------|---------|---------|
| **Call（看涨期权）** | 我付钱买个"未来涨价的权利" | 赌股票会涨 |
| **Put（看跌期权）** | 我付钱买个"未来跌价的保险" | 赌股票会跌 |

### Call 的例子

```
股票 AAPL 现在 $200
你买了一张 Call：行权价 $210，1个月后到期，权利金 $3/股

情况A：AAPL 涨到 $230
→ 你有权利以 $210 买入，立刻赚 $20/股
→ 扣除 $3 权利金，净赚 $17/股

情况B：AAPL 跌到 $190
→ 不行权，放弃权利
→ 亏损 = $3 × 100股 = $300（你付的权利金）
```

### Put 的例子

```
股票 AAPL 现在 $200
你买了一张 Put：行权价 $190，1个月后到期，权利金 $2/股

情况A：AAPL 跌到 $170
→ 你有权利以 $190 卖出，赚 $20/股

情况B：AAPL 涨到 $220
→ 不行权，损失权利金
```

---

## 最核心的概念：买方 vs 卖方

| | **买方 (Buyer)** | **卖方 (Seller)** |
|--|----------------|------------------|
| 权利 | 有权决定是否行权 | 没有权利，必须履约 |
| 最大亏损 | **权利金（有限）** | 理论上无限 |
| 最大盈利 | 理论上无限 | **权利金（有限）** |
| 谁更适合新手 | ✅ **买方** | ❌ 卖方风险大 |

> **记住：买期权 = 付保险费。卖期权 = 开保险公司。**

---

## 一个合约代表多少股

**1张美股期权合约 = 100股**

这就是为什么你看到期权的价格通常乘100：
- 权利金 $3/股 → 买1张合约要付 **$300**

---

## 期权vs股票的直观对比

| | 买100股AAPL（$200/股） | 买1张Call（行权价$210） |
|--|---------------------|---------------------|
| 投入 | **$20,000** | **$300**（权利金） |
| AAPL涨到$250 | 赚 $5,000 (+25%) | 赚 $4,700 (+1567%) |
| AAPL跌到$150 | 亏 $5,000 (-25%) | **最多亏 $300** |
| AAPL横盘不动 | 不赚不亏 | **亏 $300** |

**期权的魅力：** 用很少的钱撬动很大的收益（杠杆）。
**期权的代价：** 如果股票不涨，你的钱就归零了。

---

## 一句话总结

> 期权 = 付钱买一个"未来按某个价格买卖股票"的权利，不想要了最多损失权利金。
