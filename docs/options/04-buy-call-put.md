# 第14篇：Buy Call / Buy Put——最基础的期权操作

> 日期：2026-05-18 | 状态：初稿

---

**Buy Call 和 Buy Put 是期权最简单的两种操作**——作为买方，你的最大亏损就是付出去的权利金。

---

## Buy Call（买入看涨期权）

**场景：** 你非常看好某只股票近期会涨，但又不想花大价钱买100股。

### 实例

```
股票：NVDA 当前价 $800
你买1张 Call：行权价 $850，30天后到期，权利金 $10/股

成本：$10 × 100 = $1,000
```

**到期时的四种可能结果：**

| NVDA价格 | 你的操作 | 盈亏 |
|---------|---------|------|
| $900 | 行权，以$850买入然后卖$900 | 赚$5,000 - $1,000 = **+$4,000** |
| $860 | 行权 | 赚$1,000 - $1,000 = **$0（保本）** |
| $850 | 不行权 | **亏$1,000（亏光权利金）** |
| $800 | 不行权（脑子坏了才行权） | **亏$1,000** |

**规律：** 股价必须涨过「行权价 + 权利金」才能赚钱。

### 什么时候用Buy Call

| ✅ 适合 | ❌ 不适合 |
|--------|----------|
| 你非常看好某股票短期内上涨 | 你只是"觉得可能会涨" |
| 你不会因为亏掉权利金而难受 | 你押上了不能亏的钱 |
| 波动性大的股票（NVDA、TSLA） | 横盘或慢涨的股票 |

---

## Buy Put（买入看跌期权）

**场景：** 你觉得某只股票会跌，但不想做空（不想承担无限风险）。

### 实例

```
股票：TSLA 当前价 $250
你买1张 Put：行权价 $230，30天后到期，权利金 $5/股

成本：$5 × 100 = $500
```

**到期时的三种可能结果：**

| TSLA价格 | 你的操作 | 盈亏 |
|---------|---------|------|
| $200 | 行权，以$200买入然后以$230卖出 | 赚$3,000 - $500 = **+$2,500** |
| $230 | 保本点 | $0 |
| $250 | 不行权 | **亏$500** |

### 什么时候用Buy Put

| ✅ 适合 | ❌ 不适合 |
|--------|----------|
| 为持有的股票买保险（保护性看跌） | 频繁赌跌 |
| 重大利空事件前（财报、政策） | 用来代替做空 |

---

## Buy Call vs Buy Put 对比

| | Buy Call | Buy Put |
|--|---------|--------|
| 赌什么 | 赌涨 | 赌跌 |
| 最大盈利 | 理论上无限 | 股价跌到$0为止（有限） |
| 最大亏损 | 权利金（有限） | 权利金（有限） |
| 何时赚钱 | 股价超过「行权价+权利金」 | 股价跌破「行权价-权利金」 |

---

## 期权的杠杆效应

```
同样是赌NVDA从$800涨到$900（涨幅12.5%）：

买100股：投入 $80,000 → 赚 $10,000（+12.5%）
买1张Call：投入 $1,000 → 赚 $4,000（+400%）

但NVDA跌到$750：
买100股：亏 $5,000（-6.25%）
买1张Call：亏 $1,000（-100%，归零）
```

**期权是一把双刃剑。赚的时候翻倍，亏的时候归零。**

---

## 新手实操建议

```
✅ 只用闲钱的 5%~10% 买期权
✅ 选至少30天后到期的合约
✅ 选平价（ATM）或稍价外（OTM）的行权价
✅ 设好心理止损——亏50%就认输
✅ 第一次买不要超过1张合约（100股）

❌ 不要买只剩几天到期的"彩票"
❌ 不要全仓期权
❌ 不要因为"亏了"就死扛到期
```

---

## 一句话总结

> Buy Call = 赌涨，Buy Put = 赌跌。用有限的成本撬动大的收益，但归零风险很高。**只在你非常有信心的时候用，并且只用小仓位。**
