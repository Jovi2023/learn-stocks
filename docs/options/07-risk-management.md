# 第17篇：期权风险管理——怎么避免亏大钱

> 日期：2026-05-18 | 状态：初稿

---

期权最大的诱惑是杠杆，最大的陷阱也是杠杆。这篇只说风险——**怎么不翻车。**

---

## 期权最大的三个风险

### 1. 归零风险（最致命）

**所有买期权的钱，理论上都可以变成0。**

```
你花 $1,000 买了一张 NVDA Call
到期日 NVDA 没有涨到行权价以上
→ $1,000 = 0
```

**这不是"亏损"，是"全部亏光"。** 如果你的股票亏了，还能剩点。期权到期归零，一分不剩。

### 2. 时间损耗（隐形杀手）

```
你买Call的时候觉得"还有45天，时间长着呢"
但Theta每天都在吃掉你的权利金
到了最后7天，如果股价没动，你的期权已经一文不值
```

**期权不是股票，不能"等它回来"。**

### 3. 波动率收缩

```
你买了一堆Put赌财报后会跌
结果财报不错，并且市场恐慌情绪消除了
VIX下跌，你的Put就算股价没涨也亏了
（因为时间+波动率双重打击）
```

---

## 新手绝不能碰的

| ❌ 不要做 | 风险 |
|----------|------|
| **裸卖Call（Naked Call）** | 理论上亏损无限，别碰 |
| **末日期权**（几天内到期） | 90%归零，纯赌博 |
| **全仓期权** | 一次错了就归零 |
| **用期权补仓摊平** | 越补亏越多 |
| **买深度价外期权**（便宜但很远） | 看起来便宜，其实更难赚钱 |

---

## 期权仓位管理的铁律

### 规则一：单次投入不超过总资金的 2~5%

```
总资金 $10,000
买期权的单笔上限 = $200~$500
```

亏了这笔钱不影响你的整体投资。

### 规则二：止损要设在买入价的 30%~50%

```
你花 $1,000 买Call
当它跌到 $500~$700 时 → 卖出止损
别等到它变成 $0
```

**止损不是"亏了认输"，而是"把剩下的资金拿去下次更好的机会"。**

### 规则三：不要连续追

```
你亏了一次 → 别想着"下一把就翻回来"
越追越输，这是赌徒心理
```

---

## 到期日怎么选才安全

| 你的计划 | 建议的期权到期 |
|---------|--------------|
| 赌财报后大涨 | **财报后至少留2~3周** |
| 看好未来1个月走势 | **买45~60天到期** |
| 做Covered Call | **卖30~45天到期** |
| 做Cash-Secured Put | **卖30~45天到期** |

**通用规则：**
- 买方：至少留30天以上
- 卖方：卖30~45天刚好（Theta在最后30天加速收租）

---

## 期权风险与股票风险对比

| 风险维度 | 买100股股票 | 买1张期权 |
|---------|-----------|----------|
| 最大亏损 | 股价跌到$0（亏100%） | 权利金归零（亏100%） |
| 亏损速度 | 慢，可以等 | 快，有到期日 |
| 流动性 | 好，随时卖 | 差，价外期权难卖 |
| 迷之自信 | 少 | 多（杠杆让人盲目） |

---

## 一句话总结

> **期权是一把刀。用来切菜（Covered Call/CSP）是工具，用来切自己（裸卖/末日期权）是找死。仓位管理 + 止损 + 不追 = 你能活下去。**
