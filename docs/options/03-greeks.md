# 第13篇：希腊字母入门——知其然不用知其所以然

> 日期：2026-05-18 | 状态：初稿

---

当别人聊"Delta 0.7、Theta 被吃掉"的时候，你不需要全懂，但要知道他们在说什么。

**希腊字母 = 衡量期权价格随不同因素变动的敏感度。**

---

## 你应该知道的四个

| 字母 | 代表什么 | 通俗理解 | 你需要记住的 |
|------|---------|---------|------------|
| **Delta** Δ | 股价变动$1，期权价格变多少 | "方向敏感度" | 0~1之间，越大越像直接持有股票 |
| **Gamma** Γ | Delta本身会怎么变 | "加速器" | 越接近到期日Gamma越大 |
| **Theta** Θ | 每过一天，期权价值损耗多少 | "时间损耗" | **期权的敌人** |
| **Vega** ν | 波动率变动1%，期权价格变多少 | "情绪溢价" | 市场恐慌时期权会变贵 |

---

## Delta——最重要的

**Delta 告诉你两件事：**

1. **股价涨$1，我的期权能涨多少钱**
2. **这张期权的方向有多"像股票"**

| Delta 值 | 含义 |
|---------|------|
| Delta = 0.7 | 股票涨$1 → 期权涨$0.70 |
| Delta = 0.5 | 股票涨$1 → 期权涨$0.50（典型的ATM期权） |
| Delta = 0.9 | 期权已经很像持有股票了（deep ITM） |

Call 的 Delta 是正数（0~1），Put 的 Delta 是负数（-1~0）。

---

## Theta——期权持有者的头号敌人

**Theta = 每天期权价值会被"吃掉"多少。**

```
你买入一张期权，权利金 $5.00，Theta = -0.08
→ 明天这张期权就值 $4.92（没别的事发生的话）
→ 后天值 $4.84
```

这就是为什么期权有时间价值——你有多少天"等着股票涨"的权利。每过一天，这个权利就少一点。

**关键规律：**
- 越接近到期日，Theta 损耗越快
- 最后30天，Theta会加速"吃"你的权利金

> **新手最常见的错误：** 买了远期期权，以为"时间还多"，结果股票一横盘，权利金被慢慢吃光。

---

## Vega——市场情绪指标

**Vega 衡量市场波动率对期权价格的影响。**

- 市场恐慌（VIX上涨）→ 所有期权变贵
- 市场平静（VIX下跌）→ 所有期权变便宜

**实操：** 财报前买期权很贵（因为市场不知道结果），财报后立刻变便宜。这就是Vega在起作用。

---

## 一个实用场景

你买入 AAPL Call（行权价$200，30天后到期），这些字母告诉你什么：

| 字母 | 值 | 怎么解读 |
|------|----|---------|
| Delta | 0.50 | AAPL涨$1 → 期权涨$0.50 |
| Gamma | 0.05 | AAPL再涨$1 → Delta变成0.55 |
| Theta | -0.06 | 每天损失$0.06的时间价值 |
| Vega | 0.10 | VIX涨1点 → 期权涨$0.10 |

---

## 新手实际用不到这些

说实话，在你真正开始买期权之前，**你几乎不需要算这些数字**。

你只需要知道：

```
✅ 股价涨跌 → 看 Delta
✅ 时间消逝 → 看 Theta（别买只剩几天的期权）
✅ 市场恐慌 → 期权变贵（Vega上升）
```

等你想认真做期权策略了，再回头看这篇也不迟。

---

## 一句话总结

> Delta = 方向，Theta = 时间损耗，Vega = 恐慌溢价。作为新手，记住"Theta是你的敌人，不要在期权上持有太久的仓位"就够了。
