# 第16篇：Cash-Secured Put——以合理价格买入股票

> 日期：2026-05-18 | 状态：初稿

---

如果 Covered Call 是"收租"，那 Cash-Secured Put 就是**"蹲点等打折"**。

---

## 什么是 Cash-Secured Put？

**定义：** 你手里有现金，卖出一张 Put，承诺"如果股价跌到某个价格，我就买入"。

**通俗理解：** 你在想"AAPL跌到$180我就买"。你可以不用干等，而是**让别人付钱给你等你**。

---

## 用工厂打比方

```
你想收购一批铝合金材料
你觉得 $8,000 就是好价格
市场价现在 $10,000

你跟供应商说：
"我放 $8,000 押金在你这里
如果1个月内价格跌到 $8,000 以下，我就用 $8,000 买你的货
如果没跌到，押金归你"

供应商说："好，我付你 $200 作为'等你的补偿'"
```

---

## 具体怎么做

```
AAPL 现价 $200
你想在 $190 买入 AAPL（你认为这个价格很值）

→ 你放 $19,000 作为准备金（$190 × 100）
→ 卖出 1 张 Put，行权价 $190，权利金 $5/股
→ 你立刻收到 $5 × 100 = $500（权利金）
```

### 两种可能结果

| 到期的AAPL价格 | 发生什么 | 你的结果 |
|--------------|---------|---------|
| **$200（没跌到$190）** | 期权过期，你拿着$500 | 赚$500，继续等下次机会 |
| **$185（跌破了）** | 你被迫以$190买入100股 | 实际成本 = $190 - $5（权利金）= **$185/股** |

**关键理解：** 你本来就打算在$190买入，现在实际成本$185，赚了$5差价+$500权利金。

---

## Cash-Secured Put 的优缺点

| ✅ 优点 | ❌ 缺点 |
|--------|--------|
| 股价跌了——你"打折"买到了想买的股票 | 股价暴涨——你只能看着它涨，自己没买 |
| 股价没跌——你白赚了权利金 | 需要准备足够的现金（100股×行权价） |
| 确定性：你只买你觉得值的价格 | 如果股价一路下跌，你接到的位置也是跌的 |

---

## 什么时候用

| ✅ 适合 | ❌ 不适合 |
|--------|----------|
| 你本来就打算买这只股票 | 你只是觉得它会跌，不想买 |
| 你想在一个特定的"便宜价"买入 | 你等不及，想立即买入 |
| 市场震荡或慢跌 | 急速上涨的行情 |
| 手上有现金没处放 | 现金有更好的去处 |

---

## Covered Call vs Cash-Secured Put 对比

| | Covered Call | Cash-Secured Put |
|--|-------------|-----------------|
| 前提 | 你必须有至少100股股票 | 你必须有现金 |
| 赚什么 | 收租金（权利金） | 收"等你的钱"（权利金） |
| 风险 | 股价下跌 | 股价大幅下跌 |
| 适合什么市场 | 震荡/慢涨 | 震荡/慢跌 |
| 关系 | 我卖Call给别人 | 我卖Put给别人 |

**两个策略本质上是同一个硬币的两面。**

---

## 实操建议

```
✅ 选你想要长期持有的优质股票
✅ 行权价设在你"非常愿意买入"的价格
✅ 选30~45天后到期（权利金较多，风险可控）
✅ 每次只卖1张合约起步
✅ 权利金入账 = 立刻到手，直接进你账户
```

---

## 一句话总结

> Cash-Secured Put = 你放现金做抵押，卖Put等股票跌到你的心理价位。跌到了就低价买入，没跌到就赚权利金。**只要你真心看好这只股票长期走势，这就是个双赢策略。**
