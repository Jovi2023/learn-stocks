# 第12篇：读懂期权合约——5个要素就够了

> 日期：2026-05-18 | 状态：初稿

---

当你打开券商App的期权链（Options Chain），看到的像是一堆乱码。

其实每份期权合约只包含5个信息：

```
AAPL   $220   Call   2026-06-19   $3.50
 │       │       │        │          │
代码  行权价  类型    到期日     权利金
```

---

## 五个要素详解

### 1. 标的资产（Underlying）

对应的是哪只股票：AAPL、MSFT、SPY 等。

### 2. 行权价（Strike Price）

你锁定买入或卖出的价格。

**判定期权值不值钱：**
- Call 行权价 < 现价 → **价内（ITM）**，有价值
- Call 行权价 = 现价 → **平价（ATM）**
- Call 行权价 > 现价 → **价外（OTM）**，只有时间价值

```
AAPL 当前 $200

行权价 $190 的 Call = ITM（价内） → 贵
行权价 $200 的 Call = ATM（平价）
行权价 $210 的 Call = OTM（价外） → 便宜
```

### 3. 类型（Type）

Call（看涨）或 Put（看跌）。

### 4. 到期日（Expiration）

期权有到期日。到期那天如果不值钱，就归零。

**到期日的规律：**
- 距离到期越远 → 权利金越贵（时间多）
- 距离到期越近 → 权利金越便宜（时间在消耗）
- **到期当天还没有行权 → 合约作废**

**新手建议：** 买期权至少选 **30天以后** 到期，给自己留足够的时间。

### 5. 权利金（Premium）

买这份期权要付的钱。

权利金由两部分组成：

```
权利金 = 内在价值 + 时间价值
```

| 组成部分 | 含义 |
|---------|------|
| **内在价值** | 如果现在行权，能赚多少钱 |
| **时间价值** | "还有机会变好的"那部分溢价 |

**举例：**
```
AAPL $200 现价
行权价 $190 的 Call，权利金 $12

内在价值 = $200 - $190 = $10
时间价值 = $12 - $10 = $2（距离到期还有2个月）
```

---

## 实际看看期权链长什么样

以 AAPL 现价 $200 为例：

```
Call                                            Put
行权价  6月到期  7月到期                行权价  6月到期  7月到期
$190    $10.50   $13.00                 $190    $0.50    $1.50
$200    $3.00    $5.50                  $200    $3.00    $4.50
$210    $1.00    $2.50                  $210    $10.00   $12.50
```

**能看出什么？**
- 行权价离现价越远 → Call越便宜，Put越贵（反之亦然）
- 到期越远 → 所有期权越贵（时间价值更多）

---

## 新手操作指南

当你准备买一张期权，按顺序确认：

```
□ 哪只股票？
□ Call 还是 Put？
□ 哪个行权价？（ATM或稍微OTM，新手别选deep ITM）
□ 到期日？（至少30天后）
□ 权利金多少？我最多亏多少？
```

---

## 一句话总结

> 期权合约 = 股票 + 行权价 + Call/Put + 到期日 + 权利金。
> 改变其中任何一个，价格都会变。
