# 第18篇：Paper Trading实战——不花钱练手

> 日期：2026-05-18 | 状态：初稿

---

**在你拿真金白银买入第一张期权之前，先用模拟盘练手。**

Paper Trading = 模拟交易，用虚拟资金在真实市场环境下买卖股票和期权。**亏了不心疼，赚了也别当真。**

---

## 为什么一定要先模拟

| 现实 | 模拟盘（Paper Trading） |
|------|----------------------|
| 亏$1,000会很心疼 | 亏就亏了，当交学费 |
| 忍不住频繁交易 | 可以在模拟盘"感受"操作的冲动 |
| 不懂期权链 | 模拟盘上慢慢看懂 |
| 不敢测试策略 | 模拟盘先跑一遍 |

**你学开车也得先上驾校，而不是直接上路。美股期权也一样。**

## 模拟盘该做什么

### 第一阶段：熟悉操作（1~2周）

```
□ 找到股票代码（AAPL、SPY）
□ 下一笔市价单
□ 下一笔限价单
□ 查看持仓和盈亏
□ 找到期权链（Options Chain）
□ 看懂期权的行权价和权利金
□ 下一笔Buy Call试试
```

### 第二阶段：测试策略（2~4周）

```
□ 买VOO，持有2周不动（体验定投感觉）
□ Buy Call看涨某股票（观察Theta损耗）
□ 做一次Covered Call（模拟持有+卖Call）
□ 做一次Cash-Secured Put（模拟卖Put等买入）
```

### 第三阶段：模拟实盘（1个月）

```
□ 用$5,000虚拟资金，模拟你的真实计划
□ 严格按你的计划操作（仓位管理+止损）
□ 月底复盘盈亏和操作
```

## 模拟盘的注意事项

| ⚠️ 模拟盘骗了你什么 | 真实情况 |
|-------------------|---------|
| 模拟盘成交很快 | 真实市场可能有滑点（Slippage） |
| 模拟盘你不会恐惧 | 真金白银会让人犹豫和恐慌 |
| 模拟盘没有情绪 | 真钱会让你睡不着 |
| 模拟盘所有策略都能做 | 真实券商有权限限制 |

**所以：** 模拟盘过了以后，**第一步实盘用小钱**。$500~$1,000就够了。

---

## 从模拟到实盘——何时"毕业"

当你满足以下条件，可以考虑实盘：

```
✅ 连续2周模拟交易都是正收益（或者盈亏可控）
✅ 你能读懂期权链上的5个要素
✅ 你体验过"期权归零"（在模拟盘上）
✅ 你有严格的风险管理计划
✅ 你不会因为亏钱而影响生活
```

---

## 实操任务清单

> 打开你的模拟盘，今天就开始做：

```
第1天：直接买1张Call（随便选一个你认识的股票）
第2天：看看它涨了/跌了，感受Delta在起作用
第3天：再过一天，看看Theta吃了多少
第7天：看看有没有涨过行权价
第14天：总结一下，Call是有多难赚到钱
第30天：到期了，看看归零了还是赚了
```

**大概率你会观察到：** 到期归零的概率远高于赚钱。

这就是期权——**你付的是时间 + 波动率溢价，需要股票朝你想要的方向大幅移动才行。**
