# 第10篇：小资金实操指南

> 日期：2026-05-18 | 状态：初稿

---

**这是你读这个系列以来最实用的一篇。** 假设你有 **$1,000~$5,000** 起步，怎么做？

---

## 起步姿势

| 资金量 | 策略 |
|--------|------|
| $1,000 | **全买 VOO**（约2股）。先入场，再说别的。 |
| $3,000 | $2,000 VOO + $1,000 选1~2只你看好的大盘股 |
| $5,000 | $3,000 VOO + $2,000 分到3~4只个股 |

**核心逻辑：** 小资金最忌讳的是"分散到10只股票，每只100块"，这样涨跌都没感觉，而且还付一堆佣金。

---

## 小资金不能做的

| ❌ 不能做 | 原因 |
|----------|------|
| 买单价$1以下的股票 | 你以为便宜，其实是垃圾 |
| 买期权 | 小资金买期权 ≈ 赌博 |
| 频繁交易 | 每次$1~$2的佣金，10次就吃掉20块 |
| 日内交易 | 你的资金不够PDT门槛（$25,000） |

---

## 实操步骤

### 第一步：先买 VOO/SPY

```
无论你想买什么股票，先打底仓：
→ 账户金额的 50%~80% 直接买 VOO
→ 剩下的用来学习和试错
```

VOO 近10年年化回报约 **12%~13%**。只买 VOO，你就已经跑赢了通胀和大部分理财产品。

### 第二步：用剩下的钱"练手"

```
剩下的 $500~$1,000 ：
- 挑1~2只你研究过的公司
- 每只买 $200~$500
- 持有至少3个月，观察股价走势
- 把"为什么涨/为什么跌"记下来
```

**这不是为了赚大钱，是为了学习。**

### 第三步：不加仓不抄底

跌了就跌了，**不要补仓摊平**。补仓是华尔街用来骗人的逻辑：

- 你$100买了，跌到$50
- 你补仓$50，平均成本$75
- 你觉得"只要涨回$75就回本了"
- 但它跌到$30呢？你亏得更多

**承认错误，别让情绪绑架你的钱包。**

## 你的第一个 $10,000 计划

```
假设每个月省下 $500
1月  $500 → VOO
2月  $500 → VOO
3月  $500 → VOO
4月  $500 → VOO
5月  $500 → VOO
6月  $500 → 开始研究第1只个股
7月  $500 → VOO
8月  $500 → VOO
...累计算下来
第20个月 → 你的投资账户 ≈ $10,000
         → 其中 VOO $8,000 → 股
         → 其中 个股 $2,000（3~4只）
```

**你不需要一夜暴富。** 你只需要每个月定投，然后等。

## 最重要的是这个

```
✅ 先入场，别等"最佳时机"——没有人能抄到底
✅ VOO 打底，个股练手
✅ 不动用生活费，亏了不心疼
✅ 不要频繁交易，不要追涨杀跌
✅ 持有至少1年再评估
```

没有人是靠一笔交易发财的。但很多人在股市里活了下来，靠复利变成了富人。
