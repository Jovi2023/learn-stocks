# 第9篇：风险管理——活下来比赚得多更重要

> 日期：2026-05-18 | 状态：初稿

---

**你在股市里最先要学会的不是怎么赚钱，而是怎么不亏光。**

你家工厂的教训同样适用：**接了太多订单却周转不开，就是死路一条。**

---

## 铁律一：永远不要全仓一只股票

**你工厂只有一台关键设备。这台设备坏了，整个工厂停产。你会不会只靠一台设备？**

在股市里，all-in 一只股票就是这种风险：

| 仓位 | 出了问题 |
|------|---------|
| 全仓1只票 | 这票跌50%，你总资产亏50% |
| 分4只票 | 1只跌50%，总资产只亏12.5% |

**规则：** 单只股票不超过总仓位的 **20%**（新手建议 **10%**）。

---

## 铁律二：设止损

**止损 = 提前设定一个价格，跌到它就自动卖出。**

比如你在 $100 买入 AAPL，设止损在 **$90**（跌10%就卖）。

| 场景 | 结果 |
|------|------|
| AAPL 跌到 $90 → 自动卖出 | 你亏了10%，活下来了 |
| AAPL 跌到 $85 后反弹回 $150 | 你卖了，错过了后面50%的涨幅 |

**止损最大的敌人是：** 卖了之后它涨回来。但你可以重新买入。

**不存在完美策略。止损是用"小亏"来避免"大亏"。**

---

## 铁律三：仓位管理——用%来思考

不要想"买多少股"，而是想"投入总资金的百分之几"。

```
总资金 $10,000

买入 AAPL 10% = $1,000
买入 MSFT 10% = $1,000
买入 VOO 50% = $5,000
现金 30% = $3,000（子弹留着等机会）
```

**规则：** 永远保留一定比例的现金。大跌时你才有子弹补仓。

---

## 不要做哪些事（反面清单）

| ❌ 不要做 | 为什么 |
|----------|--------|
| 追涨杀跌 | 涨了才觉得好 → 买在顶上；跌了怕 → 卖在底部 |
| 加杠杆（融资） | 可以让你翻倍，也可以让你归零 |
| 全部买期权 | 期权自带杠杆，新手拿期权当股票买是找死 |
| 听消息炒股 | "我有内幕" → 你大概率是接盘的人 |
| 加仓摊平亏损 | 跌了不承认错误，反而补仓 → 亏得更多 |

---

## 你能承受多大的亏损？

**在做任何一次买入之前，先问自己：**

```
如果这笔钱明天变成0，我的生活受影响吗？
```

| 答案 | 说明 |
|------|------|
| 完全不受影响 ✅ | 这笔钱可以拿去投资 |
| 会有点紧张 | 减少仓位 |
| 会睡不着觉 | **先别投，攒够闲钱再来** |

---

## 一句话总结

> **不all-in、设止损、留现金、不加杠杆。做到了这四条，你就已经跑赢了80%的散户。**
