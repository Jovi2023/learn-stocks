# 第7篇：行业板块分析——哪些板块值得关注

> 日期：2026-05-18 | 状态：初稿

---

美股按行业分成11个板块（Sector），每个板块在经济周期的不同阶段表现不同。

你不一定要每个板块都懂，但要知道**现在市场在追捧什么、抛弃什么**。

---

## 11个行业板块一览

| 板块 | 代表公司 | 特点 |
|------|---------|------|
| 🖥 **科技** | AAPL、MSFT、NVDA | 高增长、高波动、高估值 |
| 🏥 **医疗保健** | UNH、JNJ、LLY | 防御性强，老龄化利好 |
| 🏦 **金融** | JPM、V、GS | 跟着利率和信贷周期走 |
| 🛒 **非必需消费** | AMZN、TSLA、HD | 经济好时大涨，经济差时大跌 |
| 🥫 **必需消费** | PG、KO、WMT | 不管经济好坏都得买，最抗跌 |
| ⛽ **能源** | XOM、CVX | 跟油价走，周期性很强 |
| 🏭 **工业** | CAT、BA、GE | 和制造业、基建直接相关 |
| 📡 **通信服务** | GOOGL、META、NFLX | 广告收入+订阅收入 |
| 🏡 **房地产** | PLD、AMT | 跟着利率走，高息时表现差 |
| 🔬 **材料** | LIN、FCX | 受大宗商品价格影响 |
| ⚡ **公用事业** | NEE、DUK | 最稳的板块，股息高，增长慢 |

---

## 新手重点关注哪些

**不用11个都研究，先关注这几个：**

- **科技** — 美股的发动机，最容易赚钱也是最容易亏的板块
- **标普500指数基金(SPY)** — 11个板块全包了，一劳永逸
- **医疗保健** — 老龄化趋势，长期刚需
- **必需消费** — 经济下行时的避风港

---

## 怎么看哪个板块当下强势

用 **Sector Rotation（板块轮动）** 的逻辑：

| 经济阶段 | 领涨板块 |
|---------|---------|
| 复苏早期 | 金融、工业、材料 |
| 扩张期 | 科技、非必需消费 |
| 繁荣晚期 | 能源、材料 |
| 衰退/防御 | 医疗、必需消费、公用事业 |

**怎么做：** 每个月花30秒到 `finviz.com` → Groups → 扫一眼"一个月涨跌幅排名"，就知道钱在流向哪些板块。
