# 第6篇：价值投资 vs 成长投资——你适合哪种

> 日期：2026-05-18 | 状态：初稿

---

你可以简单地把股票分为两类：**现在就能赚钱的**和**未来能赚大钱的**。

| | 价值投资 | 成长投资 |
|--|---------|---------|
| **赚什么钱** | 公司现在就很赚钱，买得便宜，等市场发现它的价值 | 公司还在快速发展期，看好它未来赚很多钱 |
| **典型特征** | 低PE、高股息、稳定的业务 | 高营收增长、高PE、可能还没盈利 |
| **代表公司** | 伯克希尔、可口可乐、强生 | 英伟达、Tesla、早期的亚马逊 |
| **需要什么** | 耐心、分析财务报表的能力 | 判断行业趋势的眼光 |
| **风险** | 股价可能长期不涨（价值陷阱） | 增长不如预期，PE崩塌（戴维斯双杀） |

---

## 用你家工厂来理解

**价值投资：** 你的工厂已经稳定运行10年，每年赚100万，市场给你的估价是800万（PE=8倍）。你觉得太低了——这个厂明明值1200万。于是你买下来等着。

**成长投资：** 你开了一家新的户外家具厂，专门做智能遮阳伞（带太阳能板），今年只赚了10万，但市场觉得明年能赚200万，所以愿意花$3000万买你的厂（PE=300倍）。你买的是"预期"。

---

## 哪个更好？

**没有绝对的好与坏，只有适合不适合。**

| 你的情况 | 更适合 |
|----------|--------|
| 求稳、不想看盘、准备持有3年以上 | **价值投资** |
| 想抓高增长、能接受较大波动 | **成长投资** |
| 资金量小、想快速积累 | **成长投资**（但风险也大） |
| 资金量大、不愿意亏回去 | **价值投资 + 指数ETF** |

**巴菲特名言：** "用合理的价格买入好公司，胜过用便宜的价格买入普通公司。"

---

## 新手建议

**不要二选一，两个都要。**

- **80% 的钱**放在指数ETF（如SPY、QQQ）— 自动帮你平衡价值和成长
- **20% 的钱**用来买你看好的个股，按你自己风格选

等你交易了一段时间，自然知道自己是哪种投资人。
